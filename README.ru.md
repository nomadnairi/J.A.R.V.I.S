## О проекте

J.A.R.V.I.S. — это open-source фреймворк для создания персонального AI-ассистента:
интеллектуальное ядро на базе LLM, система плагинов/навыков, вызов инструментов
(function calling) и слоистая архитектура, рассчитанная на рост в сторону
голоса, умного дома и автоматизации.

> **Статус проекта:** ранняя разработка. Ядро уже готово и работает — async-движок,
> интеграция с LLM (Anthropic / OpenAI), система навыков и инструментов,
> потоковые ответы, интерактивный CLI и **система памяти** (постоянная история +
> семантический recall). Голос, интеграции, автоматизация и web/API — в планах.
> Точный статус см. в разделе [Роадмап](#роадмап).

## Что уже работает

- **🧠 Ядро LLM** — провайдер-независимый клиент для **Anthropic (Claude)** и
  **OpenAI (GPT)** с автоматическими повторами и переключением между провайдерами.
- **🔧 Вызов инструментов (function calling)** — агентный цикл, где модель сама
  вызывает инструменты (навыки), выполняет действия и отвечает по результатам.
- **🧩 Система навыков/плагинов** — детерминированная обработка частых запросов
  (дата/время, калькулятор, диагностика системы) без обращения к LLM.
- **🧠 Память** — постоянная история диалогов (переживает перезапуск) плюс
  семантический recall (RAG): LLM извлекает **устойчивые факты** из каждого хода
  и позже вспоминает нужные. Async, на SQLite, с порогом релевантности и весом
  свежести; подключаемые эмбеддинги (offline / локальные / OpenAI).
- **⚡ Стриминг** — потоковые ответы по токенам в CLI.
- **💬 Telegram-бот** — общение с ассистентом в Telegram; у каждого пользователя
  своя постоянная сессия и память. Локализованный интерфейс (английский /
  русский / узбекский) с меню команд и выбором языка; ассистент отвечает на
  выбранном языке.
- **🎙 Голос (в боте)** — пришлите голосовое: оно распознаётся через OpenAI
  Whisper API, ассистент отвечает и (по желанию) проговаривает ответ через
  OpenAI TTS. Мультиязычно: отвечает на том языке, на котором вы говорили.
- **🔌 Интеграции** — внешние сервисы как инструменты для LLM: **погода**
  (Open-Meteo, бесплатно, без ключа) и **умный дом** (Home Assistant).
  Подключаемый каркас (connect/health/мост-в-инструменты) — легко добавлять ещё.
- **👥 Мультисессии** — множество независимых диалогов через менеджер сессий.
- **📡 Событийная архитектура** — внутренняя шина pub/sub с пассивной телеметрией.
- **🖥️ Интерактивный CLI** — чат плюс `/skills`, `/stats`, `/state`, `/reset`.

Всё перечисленное работает локально; команды навыков/инструментов работают даже
без API-ключа.

## Структура проекта

```
jarvis/
├── __main__.py        # интерактивный CLI (python -m jarvis)
├── config/            # типизированные настройки и константы
├── core/              # движок, DI-контейнер, pipeline, состояние, сессии
├── llm/               # провайдер-независимый клиент + Anthropic/OpenAI + tools
├── skills/            # система навыков/инструментов + встроенные
├── events/            # событийная шина pub/sub
├── telemetry/         # сбор метрик
├── interfaces/        # Telegram-бот (CLI — в __main__.py)
├── i18n/              # локализация (en/ru/uz)
├── models/            # Message/Conversation, Request/Response
├── memory/            # постоянная история + семантический recall (SQLite + векторы)
├── integrations/      # контракты (реализация в планах)
└── utils/             # логирование, retry, тайминги, исключения, текст
tests/                 # набор тестов pytest
docs/                  # документация по архитектуре
```

## Быстрый старт

**Требования:** Python 3.10+

```bash
# Клонирование
git clone https://github.com/nomadnairi/J.A.R.V.I.S.git
cd J.A.R.V.I.S

# Установка
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Настройка (добавьте свой API-ключ)
cp .env.example .env            # укажите ANTHROPIC_API_KEY или OPENAI_API_KEY

# Запуск
python -m jarvis
```

Предпочитаете `make`? `make install`, `make run`, `make test`, `make lint`.

## Использование

В CLI:

```
Sir › calc (12.5/100)*320
J.A.R.V.I.S. › (12.5/100)*320 = 40

Sir › what time is it
J.A.R.V.I.S. › It is 14:05.

Sir › system status
J.A.R.V.I.S. › All systems nominal. …

Sir › /skills      # список навыков и какие доступны LLM как инструменты
Sir › /stats       # телеметрия сессии
Sir › /memory      # статистика памяти
Sir › /reset       # очистить диалог (долговременная память сохраняется)
Sir › /forget      # стереть историю и долговременную память
```

Всё, что не обработал навык, отвечает LLM (при необходимости вызывая инструменты).

## Telegram-бот

Общайтесь с J.A.R.V.I.S. в Telegram. У каждого пользователя своя постоянная
сессия, поэтому ассистент помнит каждого отдельно.

```bash
pip install aiogram            # опциональная зависимость интерфейса

# В вашем .env:
#   TELEGRAM_BOT_TOKEN=...      (от @BotFather)
#   ANTHROPIC_API_KEY=...       (или OPENAI_API_KEY)

python -m jarvis.interfaces.telegram_bot     # или: jarvis-bot
```

Команды: `/language` (сменить язык интерфейса и ответов), `/reset` (очистить
текущий диалог), `/forget` (стереть всё, что бот о вас помнит), `/help`. Меню и
интерфейс локализованы на **английском, русском и узбекском**, ассистент
отвечает на выбранном пользователем языке. Доступ можно ограничить конкретными
user ID через `TELEGRAM_ALLOWED_USERS`.
