# KER v1.9.2

A desktop polish release: the app itself now carries the KER name everywhere.

## ✨ Fixes

- **Desktop app fully rebranded to KER.** The sign-in dialog, window title,
  tray, notifications, chat and message boxes no longer show the old name —
  they all read **KER** now (EN / RU / UZ).
- **Auto-updater fixed.** It now downloads `KER-Setup.exe` (matching the
  renamed installer) instead of the old filename.

## ℹ️ Notes

- To run the desktop app standalone with the live dashboard, choose
  **"Continue locally"** on the sign-in screen — the bundled engine then serves
  the Command Deck live. "Account (server)" mode expects a running KER server;
  if none is reachable, the dashboard shows the offline demo state.
