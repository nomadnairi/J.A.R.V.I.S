# KER v1.9.4

A redesigned Home — the Command Deck now opens on a three-column workspace.

## ✨ What's new

- **New 3-column Home.** Inspired by a premium desktop layout, the home screen
  is now a real workspace:
  - **Left** — quick-access cards to every screen, a live **System** panel
    (provider, model, memory, internet, API) and a KER info card with the
    animated reactor + real Python / uptime / sessions / mode.
  - **Middle** — a compact live **model catalogue** (popular / free / premium,
    with search) drawn from the real registry.
  - **Right** — a working **chat** panel wired straight to the engine.
- **All real data.** Every field comes from the backend; nothing is faked.
- Everything from v1.9.3 (honest telemetry, WebGL 3D reactor) carried forward.
